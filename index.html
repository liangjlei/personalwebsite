Jinglei Liang

Embodied AI · M.Sc. Student, CUHK

I am an M.Sc. student in Computer Science at The Chinese University of Hong Kong (CUHK), where I work on embodied intelligence. I received my B.Eng. in Computer Science from Sun Yat-sen University (SYSU).

My research interest lies in vision–language–action (VLA) models and world & action models (WAM) — end-to-end embodied policies that learn to perceive, reason, and act in the physical world. I am particularly concerned with closing the perception–manipulation loop, marrying large multimodal models with reinforcement learning for robust closed-loop control on real robot hardware.

Most recently, I led the on-device speech & manipulation pipeline for a live human–robot interaction demo, attaining 95%+ intent accuracy and an 80.4% place success rate across 110+ on-robot trials.

Selected Projects

A selection of my work in embodied AI — robot perception, speech interaction, vision–language–action policies, and the agent systems around them. Representative projects are highlighted. Each links to a full write-up.

Streaming Multimodal Speech Grounding for Real-Time Human–Robot Interaction
Jinglei Liang
Technical Report, 2026
project page / report

An end-to-end voice-to-action stack for service robots: mic PCM → VAD → ASR → NLU → VLM/SAM → motion planner → TTS, fully decoupled as ROS2 nodes under a state-machine orchestrator. I rebuilt the streaming-ASR path (stable 150–300 ms partials), designed a three-stage voice-activity gate that cut inference compute by 86% and tripled far-field recall, and moved color grounding from HSV heuristics into the vision–language stage to eliminate referent mis-rejections.

Parameter-Efficient Adaptation of Vision–Language–Action Policies for Long-Horizon Manipulation
Jinglei Liang
Preprint, 2026
project page / report

A five-component recipe for parameter-efficient VLA adaptation: selective LoRA placement, LLM-based instruction augmentation, test-time action-chunk fusion, EMA weight averaging, and failure-case resampling. Together they lift mean success from 92.2% to 94.2% and the long-horizon split to 88.8% (+3.6 pt over the official checkpoint), while cutting flow-matching loss by 80% over 2,000 evaluation episodes.

Aloha π0.5 LeRobot: A Split-Stack Pipeline for Real-Robot Teleoperation, Training, and Remote Inference
Jinglei Liang
Open-source system, 2026
project page / code / demo video

An end-to-end real-robot learning stack that adapts Hugging Face LeRobot into a distributed Aloha manipulation pipeline. AlohaMini leader arms drive bimanual teleoperation; multi-camera demonstrations are recorded, cleaned, and merged in LeRobot format; π0.5 vision–language–action policies are fine-tuned on an A100 (bf16 + gradient checkpointing); and large checkpoints are served from a remote RTX 4090 while the Orin robot host keeps camera capture, action validation, watchdogs, and a safety gate close to the hardware over a closed-loop gRPC link.

Neural Motion Retargeting for Affective Whole-Body Teleoperation
Jinglei Liang
Technical Report, 2026
project page / report

A “perceive → infer affect → generate motion → drive hardware” interaction system. I replaced analytic IK with T-pose calibration by a 4-layer Transformer retargeting model (1.0 M params, 60-D IMU → 10-D encoding), markedly improving motion smoothness and robustness, and added a two-stage detection + vision–language perception cascade that dropped per-frame latency from 1.5 s to ~920 ms.

A Comparative Study of Audio–Visual Active Speaker Detection in Unconstrained Scenes
Jinglei Liang
Survey / Technical Report, 2026
project page / report

A coarse-to-fine comparison of three audio–visual paradigms — semantic audio–visual segmentation, audio–visual instance segmentation, and active speaker detection — for the “who is speaking to me” problem in crowded scenes. I reproduced and benchmarked all three and selected a two-stage detect → track → ASD pipeline (model < 20 MB, real-time) along task alignment, accuracy, latency, and deployability.

Robust Hand–Eye Calibration via SE(3) Maximum-Likelihood Estimation
Jinglei Liang
Open-source toolkit, 2026
project page / guide

A standardized hand–eye calibration pipeline (AprilTag 36h11, maximum-likelihood estimation with Gauss–Newton on SE(3) under a Huber robust loss, millimeter-level residuals) plus a reusable intrinsics procedure covering both eye-on-base and eye-on-hand configurations. The toolkit drove a long-running automated assembly demonstration with 20/20 consecutive successful runs across 10+ public showings.

A Multi-Tenant Control Plane for Orchestrating Large Language Model Agents
Jinglei Liang
Systems Report, 2026
project page / overview

A control plane that turns a bare agent runtime into a team-ready platform. I built the orchestration layer (Hono + SQLite): multi-user auth (API key / OIDC) with audit logging, a unified streaming invoke() routing across four model families behind one API, and per-tenant isolation of skills, knowledge, and shared memory, bridged to repository sync and webhooks for end-to-end delivery.

Toward an Internet of Agents: Structured Tool Connectors for Autonomous Catalog Reasoning
Jinglei Liang
Open-source project, 2026
project page / code

A lightweight connector layer that lets autonomous agents query an external catalog through a single structured protocol. Agents fetch item details, search, and inventory via a brand-agnostic JSON API with rate-limited, retry-safe access — turning an opaque third-party data source into a typed tool that planning and pricing agents can call without bespoke integration code.

Tool-Augmented General-Purpose Agents: Planning and Sandboxed Execution
Jinglei Liang
Technical Note, 2026
project page / notes

A study and extension of a ReAct-style agent that plans, selects tools, and executes them in a sandbox with a pluggable LLM backend. I analyzed the planning–tool–observation loop, added and evaluated custom tools, and used the framework as a testbed for how general agents can drive the perception and robotics services in my other projects.
