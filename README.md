# Jinglei Liang — Personal Website

Academic homepage for **Jinglei Liang**, working on embodied AI: robot
perception, speech interaction, and vision–language–action models.

- **Live site:** `index.html` (plain HTML + CSS, no build step)
- **Design:** custom layout; the Selected Projects list follows the academic
  paper-list style popularized by [Jon Barron's website](https://github.com/jonbarron/jonbarron.github.io).
- **Project write-ups:** see [`projects/`](projects/) — one README per project.
